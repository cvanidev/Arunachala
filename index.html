Arunachala

A simple responsive WordPress theme

View project onGitHub

Features

  • Simple, light and responsive design
  • 1 & 2 column flexible layout
  • Custom background
  • Custom header image and site title color
  • Custom Menu. Lightweight navigation menu for tablets and mobile devices.
  • Sticky posts slider
  • Custom grid view layout for posts on homepage
  • Widgets ready right sidebar
  • Full width image gallery with navigation and zoom
  • 8 post formats support : standard, aside, image, video, audio, quote, link and gallery
  • Localization support : Translation ready and right to left (RTL) languages stylesheet included
